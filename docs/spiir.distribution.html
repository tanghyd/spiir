draw(n: int, redraw: bool = True) → DataFrame

Draws samples from all stored Distribution objects.

If a Constraint is present, the constraint is applied and samples are rejected until n valid samples have been drawn. Follows a similar method from PyCBC: https://github.com/gwastro/pycbc/blob/master/pycbc/distributions/joint.py#L305.
